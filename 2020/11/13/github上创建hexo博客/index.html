github上创建hexo博客

安装

  1. 安装node.js

  2. 安装hexo。npm install -g hexo-cli

  3. hexo项目初始化。hexo init <新建文件夹的名称>

  4. 如果是从git上下载下来的hexo项目，安装依赖包。npm install –force

  5. 创建文章。 hexo new post fileName

  6. 发布文章。hexo d (修改配置文件，安装npm install hexo-deployer-git –save)

  7. 启动。hexo s

  8. 编译。hexo g
